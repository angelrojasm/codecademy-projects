HTML Input Types and Attributes

by: Angel Rojas

Input Types:


Input Type Description
Text

Most primitive form of catching input. Lets user manually input text into the field.

Number

Allows user to input a numeric value and also scroll through numbers

Email

Captures text and validates it to be in email form. Probably uses a regex :p

checkbox

Allows user to choose many options from a group

Radio

Allows user to choose only one option from a selected few

Select

Utilizes a dropdown list to let the user choose one from many options.

Range

Utilizes a Slider to choose a range amount between options



Attributes:

Attribute: Belongs To: Description:

required

 All This attribute makes it so that the specified input must be filled out

pattern, minlength, maxlength

 Text These attributes are used to set the constraints on the text input provided by the user.

min, max, step-size

 Range,Number These attributes are used to set the range of numbers available for input and the amount of slide the slider uses.
